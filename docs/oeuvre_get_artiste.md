## \[GET\] oeuvre/findByArtiste/\{id_artiste\}

### Description

Récupérer toutes les oeuvres d'un artiste.

### Requête

| Paramètres       |Type      ||
| ------------- | :-----------: | -----: |
| id_artiste      | int | obligatoire |


### Réponse

| Paramètres       |Type      |
| ------------- | :-----------: |
| id      | int |  
| titre    |   string    |
| date    |   int    |
| img     |   string    |
| anecdote     |   string    |
| mouvement     |   string    |
| artiste     |   string    |
