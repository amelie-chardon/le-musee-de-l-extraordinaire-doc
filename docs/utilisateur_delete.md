## \[DELETE\] utilisateur/

### Description

Supprimer le compte et les informations associés de l'utilisateur connecté.
Cette fonction permet de supprimer uniquement son propre compte.


### Requête

| Paramètres       |Type      ||
| ------------- | :-----------: | -----: |
| token (header)      | string | obligatoire |


### Réponse

| Paramètres       |Type      |
| ------------- | :-----------: |
| status      | bool |  
| error     |   string    |
| message     |   string    |
