## \[GET\] Utilisateur/

### Description

Récupérer les informations de l'utilisateur connecté.
Cette fonction permet de récupérer uniquement ses propres informations.


### Requête

| Paramètres       |Type      ||
| ------------- | :-----------: | -----: |
| token (header)      | string | obligatoire |


### Réponse

| Paramètres       |Type      |
| ------------- | :-----------: |
| id      | int |  
| identifiant      |   string    |
| email |   string    | 
| role      | int |
